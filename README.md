# 제목
(이미지 들어갈 수 있으면 좋음)
프로젝트 간단 설명(1~2줄 정도)

## 📥 설치
```bash
npm install
```

## 💿 실행
### 서버 실행
```bash
npm run server
```
### 클라이언트 실행
```bash
npm run dev
```
